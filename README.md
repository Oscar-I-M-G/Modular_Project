# Modular_Project

Este repositorio contiene todos los documentos y recursos relacionados con el proyecto modular: un brazo robótico pequeño. A continuación, se describe la estructura y los componentes del proyecto.

## Estructura del Proyecto

- **Documentación**  
  Aquí se encuentra un proyecto en LaTeX que será entregado al profesor. Incluye:
  - Introducción al proyecto
  - Diseño y especificaciones técnicas
  - Análisis de componentes
  - Resultados y conclusiones

- **Archivos 3D**  
  Contiene modelos 3D necesarios para la construcción del brazo robótico. Incluye:
  - Piezas imprimibles en 3D
  - Archivos CAD (e.g., `.stl`, `.step`, `.f3d`)
  - Instrucciones de ensamblaje

- **Electrónica**  
  Incluye todo lo necesario para la parte electrónica del proyecto:
  - Esquemáticos de PCB
  - Lista detallada de materiales electrónicos (BOM: Bill of Materials)
  - Archivos de diseño de PCB (e.g., `.pcb`, `.sch`, `.kicad_pcb`)

- **Código**  
  Código fuente utilizado para controlar el brazo robótico. Incluye:
  - Controladores de motores
  - Algoritmos de movimiento
  - Interfaz (si aplica)
